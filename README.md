# transferred

The most convenient data transfer tool.

Pre-0.0.1. See [PLAN.md](./PLAN.md) for roadmap and [DESIGN.md](./DESIGN.md) for architecture.

## Crates

- `transferred-core` — connector-agnostic traits and types.
- `transferred-parquet` — local Parquet source and destination.
- `transferred-postgres` — Postgres source (scaffold; full impl in 0.1.0).
